COMP 2156 – DevOps for System Administration is a course focused on modern software deployment and automation practices.


DevOps is a combination of Development and Operations. It promotes collaboration between software developers and IT operations teams to improve software delivery speed and reliability.

Students in this course learn how to collaborate using GitHub and manage versions using Git. Crucial abilities acquired during the course include branching, merging, pull requests, and commit management.

Additionally, Continuous Integration (CI) and Continuous Deployment (CD) are introduced throughout the course. By automating testing and deployment procedures, these techniques lower human error and boost productivity.

In order to automate operations, students have practical experience using GitHub Actions and other CI/CD tools.

Concepts related to Linux system administration, such as file management, permissions, and command-line operations, are also discussed.

COMP 2156's primary objective is to equip students with practical skills in automation, teamwork, and infrastructure management in order to prepare them for DevOps careers in the real world.
